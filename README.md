Student Grade Calculator

About Project
This is a simple Python project that takes student marks and calculates total, percentage, and grade. It also saves result in a file.

Features
- Student name input
- Marks input
- Total calculation
- Percentage calculation
- Grade system (A+, A, B, C, D, F)
- Save result in result.txt

Grade System
90 to 100 = A+
80 to 89 = A
70 to 79 = B
60 to 69 = C
50 to 59 = D
Below 50 = F

How to Run
python student_grade.py

Output Example
Name: Mini
Total: 250
Percentage: 83.33
Grade: A

Save Feature
If user selects yes, result saves in result.txt
